#  MERN Stack Project

##  Overview
This project is a full-stack web application built using the **MERN Stack**:
- MongoDB (Database)
- Express.js (Backend Framework)
- React.js (Frontend Library)
- Node.js (Runtime Environment)

It demonstrates how frontend and backend work together to create a complete web application.



##  Tech Stack

### Frontend
- React.js
- HTML5, CSS3, JavaScript
- Axios (for API calls)

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)
