# 👩‍💻 Employee Management REST API (Backend)

Developed by: **Vijaya Lakshmi**

This is a Node.js and Express backend built to manage Employee records using a MongoDB database. It provides a complete set of CRUD (Create, Read, Update, Delete) endpoints.

## ⚡ Features

- Node.js + Express: Fast backend server  
- MongoDB + Mongoose: Database handling  
- ES Modules: Modern JavaScript (import/export)  
- dotenv: Environment configuration  
- CORS Enabled: Connects with frontend  

---

## 📌 Prerequisites

- Node.js installed  
- MongoDB (Local or MongoDB Atlas)
